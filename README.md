## Installation
### For Regular Users
1. Download and extract the archive (lossless scaling free download)
2. Run LosslessScaling.exe as Administrator
3. Follow the setup guide to install Lossless Scaling utility
4. Enable LSFG frame generation, AMD FSR, and performance options
5. Launch your game in windowed mode with maximum performance and start playing instantly

## Troubleshooting
| Problem | Solution |
|--------------------------------|-------------------------------------------------------|
| Utility doesn't start | Run as Administrator and disable antivirus temporarily |
| Frame generation not working | Ensure game is in windowed or borderless window mode |
| Black screen / screen flickering| Enable performance optimization or change scaling mode |
| LSFG / FSR lag issues | Update to latest version 2026 and lock your base game FPS |
| Windows 11 compatibility error | Update Windows and run the setup guide again |
| Game crashes on scale | Verify game installation and clear Lossless Scaling cache |
| Download / update failed | Use the latest release and check your internet connection |
